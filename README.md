# Glofox API Client

This is a Golang client for the Glofox API, providing endpoints for managing:

- **Classes**
  - `GET`, `POST`
- **Bookings**
  - `GET`, `POST`

## Getting Started

You can build and run the service using Docker.

### Build and Run

```bash
docker-compose up --build
```

### Test

```bash
go test -v ./...
```
